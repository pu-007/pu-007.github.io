## 一、

## 五、

说回学习

学习的本质或许可以用一个式子概括

```
输入 --内化--> 输出
```

它很简单, 人人都会, 人人也都知道, 自然天成

可我们却仍然在追寻各种各样的学习方法

什么"错题本", "笔记法", 漂漂亮亮好学生

什么"解题大招", "秒杀法", 神乎其神好老师

又或者"补课班", "这个视频让你上清华"等等等等

我们在追寻什么? 其实只是捷径, 我们本能地在"偷懒"

假如你想努力了, 像打了鸡血一样,

可过了几天, 可能又会按人本能的惯性再次"偷懒"

之后便又是自责、懊悔,

或者根本没有, 就直接摆烂吧

为什么我们明明知道、明明想努力却依然不能"自律"呢?

因为那样的"自律"实质是"控制", 是强迫自己去学习

依旧去思考, 你, 为什么要学习? 你想要什么样的学习成绩?

## 六、

"偷懒"其实是我们本能地"爱"自己,

用"恨"与"爱"作对自然是很痛苦地"自律"

所以才反复地去思考, 自己学习的原因

对于我来说,
    我希望得到他人的认可, 不被人批评, 享受到他人的青睐
    我希望可以赚更多的钱, 有更高的社会地位, 有圣人一般的境界
    
我也希望国家社会做出创新与贡献, 在历史中留下自己生命的遗产; 
    我希望可以突破自我界限, 充盈自己的思想, 有我爱的人与爱我的人

我希望有幸福、快乐地生活

尽管我们其实并不需要他人的认同,

因为我们与他人都是平等的,

他们批评或者表扬我们与我们没有任何关系,

可在身处在这个共同体中,

无论哪种方式, 你的, 或者我的, 或者我们的

最终都是希望我们自己能够获得爱与幸福

这就是我们最本质的动力

因此, 要学好习并不是痛苦地在学校熬日子,

而是勇敢地过好每一天, 勇敢地追求爱与幸福

## 七、

即使有了动力,

也需要指导行动的思想与实际行动

再看学习的原理

```
输入 --内化--> 输出
```

无论看成千上万个学习视频, 听成千上万个学习方法,

其核心都有以下四点: **思考**、**记忆**、**归纳**、**自律**

1. **思考**: 

所有的学习, 必须自己进行思考, 换句话说, 就是老师讲课你得过脑子, 做题你也得过脑子,

只有自己积极主动地思考, 学习才是有用的

否则就只是听从指令的机械, 这也就是被动学习, 被逼着学习效果并不好的原因

2. **记忆**

此处应先区分两种概念, 一个是`信息`, 比如中国省份, 一个是`知识`, 比如闰年的判断

针对不同内容, 我们应采取不同方式

很多人都将记忆与学习弄混

记忆是重新复现一个信息, 学习是可以学会一个能力,

而对于广义上的"记忆", 二者没有区分, 所以都可归为记忆

3. **归纳**

对自己的行为和方法总结复盘

对知识体系整理成框架

4. **自律**

一种生活态度

关于具体的方法, 可以参考另一篇`个人的学习方法总结`

## 八、

说了这么多, 我们日常应该怎么生活呢?

像学习机器一样, 又或者像别人家的孩子一样远离七情六欲?

其实并不是

作为学生, 学习, 自律, 就是我们的生活

我们要享受生活, 同样也要享受学习, 自律

可能听起来有点扯, 但并不痛苦

你当然可以去探求"爱"

你当然可以去做各种娱乐

你当然可以去"追星"

同样, 你也可以去好好学习

很简单的一句话,

"该学的时候学, 该玩的时候玩"

有各种欲求完全正常,

只是我们是否可以先做一些对正上高中的我们比较重要的事?

想休息想躺平也很正常,

只是我们是否可以先完成基本的任务, 然后轻松愉快的躺平?

我们都知道学习苦,

只是我们能不能思考一下我们为什么学习? 想不想让我们以后更加快乐更加幸福?

为什么非要抵触不可呢?
